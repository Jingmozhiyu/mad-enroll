### Project Structure：

![MadEnroll](./backend/MadEnroll.jpeg)

### Tech Stack

Backend：
- Java 21
- SpringBoot 4
- MySQL
- RabbitMQ
- Redis
- Docker

Frontend：
- Next.js 16
- React 19
- TailwindCSS 4
- TypeScript

### How it works 

Subscribe courses after registering by email address. Users will receive email alerts when seats become available.
